### Glossary

Back to README: [../README.md](../README.md)

- **Provider**: Flutter state management package; `ChangeNotifier`-based classes under `lib/Providers/**`.
- **Service**: Feature-focused class in `lib/Services/**` handling networking and business logic.
- **ApiBaseHelper**: HTTP client wrapper in `lib/Network/api_base_helper.dart`.
- **ApiLinks**: Endpoint path constants in `lib/Network/api_links.dart`.
- **Ekvipedia**: In-app content hub and events.
- **Ekvi Journeys**: Guided course-like content with lessons and progress.
- **ARB**: Application Resource Bundle for localization, under `lib/l10n/**`.
- **OneSignal**: Push notification provider integrated via `lib/core/di/one_signal_singleton.dart`.
- **Purchases**: In-app purchases handled via `purchases_flutter` and helpers.
